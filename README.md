# telegram_weather_bot

A Telegram bot that suggests what to wear based on your current location's weather.  
Outfit advice is generated by [Google Gemini](https://aistudio.google.com/) (`gemini-2.5-flash-lite`).  
Weather data from [Open-Meteo](https://open-meteo.com/) (free, no key).  
Supports **English** and **Ukrainian**.

## Usage

1. Send `/start` and choose your language (English / Ukrainian)
2. Tap **Share my location**
3. Tap **Check outfit**

You can also type `@botname` in any chat to get your outfit suggestion inline.

### Commands

| Command | Description |
|---|---|
| `/start` | Set up the bot (language + location) |
| `/language` | Change language |
| `/mylocation` | Show your saved coordinates |
| `/outfit` | Get an outfit suggestion now |
| `/help` | Show this command list |

## Environment variables

| Variable | Required | Where to get it |
|---|---|---|
| `BOT_TOKEN` | Yes | [@BotFather](https://t.me/BotFather) on Telegram |
| `GEMINI_API_KEY` | Yes | [Google AI Studio](https://aistudio.google.com/apikey) |

Copy `.env.example` to `.env` and fill in both values before running.

```
BOT_TOKEN=your_telegram_bot_token
GEMINI_API_KEY=your_gemini_api_key
```

## Run locally

```bash
git clone <repo-url>
cd telegram_weather_bot
python -m venv .venv && source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env   # edit .env and paste your BOT_TOKEN and GEMINI_API_KEY
python bot.py
```

`users.db` is created in the working directory on first run.

## Deploy on Railway

1. Fork / clone this repo and push it to GitHub
2. Create a new [Railway](https://railway.app/) project → **Deploy from GitHub repo**
3. Add environment variables: `BOT_TOKEN` and `GEMINI_API_KEY`
4. Railway picks up the `Procfile` and starts the worker automatically

Note: `users.db` is ephemeral on Railway unless you mount a volume. For durable persistence, swap `aiosqlite` for a Postgres connection.

## How it works

- Weather is fetched from Open-Meteo for the user's saved coordinates (current UTC hour).
- The weather data (temperature, wind speed, WMO weather code) is sent to Gemini, which returns a short outfit recommendation in the user's chosen language.
- If Gemini is unavailable, a rule-based fallback generates a basic suggestion locally.
- Outfit suggestions are cached in memory per user for 30 minutes to avoid redundant API calls.
